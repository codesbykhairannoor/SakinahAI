# SakinahAI

Prototipe aplikasi web produktivitas personal "Sakina".
Dibuat dengan React, Vite, Tailwind CSS, dan Framer Motion.

## Cara Menjalankan

1. `npm install`
2. `npm run dev`

## Deploy ke Vercel

Proyek ini sudah dikonfigurasi menggunakan Vite dan dapat langsung di-deploy ke Vercel tanpa pengaturan tambahan. Cukup hubungkan repositori GitHub ini ke Vercel dan pilih *preset* Vite.
